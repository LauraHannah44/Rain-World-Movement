# Turn-Drop
![[turndrop_header.gif]]
## Description
It is possible to [[turnstore]] by initiating a [[dashturn]] as slugcat reaches a ledge, dropping off it due to the skid momentum.

## Notes
This move widens the window to [[ledgedrop]] or [[ledgedroproll]].
